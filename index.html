Autonomous Memory Folding

Long interactions explode context length. DeepAgent autonomously decides to "take a breath" ( ). It compresses history into three structured memory types, preventing error accumulation.

  • history_edu

    Episodic Memory

    Long-term log of major milestones.

  • light_mode

    Working Memory

    Current sub-goal and immediate status.

  • build

    Tool Memory

    Usage stats and error patterns.

ToolPO: Training for Stability

Training on real APIs is unstable and expensive. ToolPO solves this with two key innovations.

LLM-Simulated APIs

Instead of hitting real, flaky APIs during RL training, an Auxiliary LLM acts as a "Tool Simulator", mimicking outputs instantly and deterministically.

Advantage Attribution

Sparse rewards (pass/fail) aren't enough. ToolPO assigns credit specifically to the tool invocation tokens that led to success, refining the "how" not just the "what".
